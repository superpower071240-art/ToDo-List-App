**DESCRIPTION OF TASK PERFOREMD** : 
The To-Do list Application is a Python - based project developed to help users manage daily tasks efficiently. 
The Application allows users to add new tasks as completed, and delete tasks when no longer needed. Task 
Data is stored locally, ensuring that tasks remain saved even after the program is closed and reopened.
The Project was designed with a simple and user - friendly interface, making it easy for users to track their work 
and retrieve task information automatically. Error handling was also included to prevent crashes caused by invalid
user inputs.
Through this project, key python concepts such as lists, functions, loops, condtional stataments, file handling , and 
exception handling were applied in a practical real - world application.
